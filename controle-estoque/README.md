# Controle de Estoque

Uma aplicação simples e intuitiva para gerenciamento de estoque, desenvolvida como parte de um projeto de aprendizado em Angular.

## 🚀 Funcionalidades

- **Autenticação:** Sistema de login e registro de novos usuários.
- **Dashboard:** Visão geral rápida do estado atual do estoque.
- **Gerenciamento de Estoque:** Listagem e controle de produtos.
- **Painel Administrativo:** Área restrita para configurações e gestão superior.
- **Segurança:** Proteção de rotas através de Guards (Auth e Role Guards).

## 🛠 Tecnologias Utilizadas

- **Angular 21:** Framework principal.
- **Angular Material:** Biblioteca de componentes para uma interface moderna.
- **TypeScript:** Linguagem base para o desenvolvimento.
- **CSS:** Estilização personalizada dos componentes.

## 🏁 Como Iniciar

### Pré-requisitos

Certifique-se de ter instalado em sua máquina:
- [Node.js](https://nodejs.org/)
- [Angular CLI](https://angular.dev/tools/cli)

### Passos para Instalação

1. Clone o repositório ou baixe os arquivos.
2. Abra o terminal na pasta raiz do projeto (`controle-estoque`).
3. Instale as dependências necessárias:
   ```bash
   npm install
   ```

### Executando o Projeto

Para rodar a aplicação localmente, utilize o comando:
```bash
ng serve
```
Após o comando, abra o seu navegador e acesse: `http://localhost:4200/`. A página será atualizada automaticamente se você fizer alterações no código.
